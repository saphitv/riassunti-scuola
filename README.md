# Riassunti Scuola

This repository contains my personal school summaries and study notes, written and organized by me.

The project is also a small website that makes the material easier to browse by subject, topic, and semester. Some summaries are already usable, while others are still work in progress and will be expanded over time.

## What This Project Is

- A personal collection of school notes and summaries
- A searchable web app for browsing the material more comfortably
- An ongoing project, not a finished or official academic resource

These notes are mainly for my own study workflow, but the repository also acts as a structured archive of the subjects I am covering.

## Current Subjects

- Analisi 2
- Numerica
- Programmazione a Oggetti
- Programmazione Procedurale
- Microcontrollori
- Probabilità e Statistica

Depending on the subject, you may find formulas, definitions, examples, code-related notes, cheat sheets, and linked reference material.

## Project Status

Some parts of the repository are complete enough to read and use, while other sections are still unfinished. I update the content progressively when I have time or when I need to review a specific topic.

## About The Code

The study content and summaries are mine.

Most of the code for the repository was built with AI assistance. AI was used mainly to speed up the development of the website, UI structure, components, and general implementation details. The project direction, content organization, and the actual school material are still managed by me.

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- MathJax for math rendering

## Run Locally

Install dependencies and start the development server:

```bash
pnpm install
pnpm dev
```

Then open `http://localhost:3000`.

Useful commands:

```bash
pnpm lint
pnpm ts:check
pnpm build
```
